    Farmburgh - Solo Game Dev

    Farmburgh is a point-and-click farming simulator that I independently developed and released on Steam. Prior to Farmburgh, I had played around in Unity and Unreal, creating simple prototypes and learning what I could. I felt that it was time to actually develop a game from start to finish in order to get a small taste of the industry. Farmburgh was developed in Unity using C# and all of the programming and game design was done by me. I quite enjoyed this process, especially coming up with creative features and overcoming difficult problems. Check out the project page to view some code samples, screenshots and where I plan to take the game next.





    Excel Quoting Template

    Not long after starting as an estimator, I quickly got fed up of how tedious it was to enter pertinent information into the old version of the quoting spreadsheet. After implementing just a few table lookups, I had managed to speed up my quoting efficiency drastically. My boss soon noticed an was thrilled. Together with my managers and the other estimators, I was able to create a whole new template in Excel, using lookups, VBA and UDF’s, to meet our needs. This had us quoting up around 100 jobs per week, up from an average of 60, in less time than it would take to quote that 60. Check out the link below to view some of the VBA code that went into this template as well as an example of the template itself (prices randomized to maintain company secrecy).














    March Madness Machine Learning Project

    This project uses logistic regression, K-nn and SVM classifications in order to predict first-round March Madness game winners. This was done as the final project for ST 442: Intro to Data Science. The data was scraped using Python and data cleaning and analysis was done in R. The project page below highlights the important information from the paper. There is also a link for the Github repo where the R code and data used can be found. All the technical jargon can be found in the report itself. Project will be revisited and revised when I get the opportunity.
